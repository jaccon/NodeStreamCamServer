# NodeStreamCam Server
This is a simple streaming server running with NodeJS, WebSocket and Express.

#### How to use
First, you need running a command in terminal to install all dependencies

`yarn` 

or 

`npm i `

#### Start the Streaming Server

`yarn start `

or

`npm start`
